# BERT Legal API

Entity extraction and summarization for US legal documents using BERT + BM25.

## Features

- ✅ US-specific legal entity extraction (cases, courts, citations, statutes)
- ✅ High-quality document summarization using BERT + BM25
- ✅ Important facts extraction
- ✅ Fast processing (3-6 seconds per document)
- ✅ Low memory footprint (1-2GB)
- ✅ RESTful API with FastAPI
- ✅ Docker containerized
- ✅ Jenkins CI/CD ready
